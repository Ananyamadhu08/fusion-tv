## Fusion TV
A video library application.

## Tech Stack - 
- React
- Fusion CSS | https://github.com/Ananyamadhu08/fusion
- MockBee for mock backend

## Features - 
- explore videos
- create playlists
- add videos to playlists
- like videos
- video history
- watch later
- filter videos by categories
- authentication
